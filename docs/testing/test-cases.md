# Brainer — тест-кейсы для ручного тестирования

Каталог тест-кейсов приложения Brainer (`app.obsidianmd`, KMP/Compose). Для каждого кейса:
название, шаги воспроизведения, ожидаемый результат, отметка о доступности автоматизации и
ссылка на Maestro-флоу (если автоматизирован).

**Легенда «Автоматизация»:**

- ✅ **Да** — есть Maestro-флоу, прогнан на эмуляторе Pixel 9a (ссылка на файл).
- ⚙️ **Доступно** — можно автоматизировать, но пока вручную (нужны фикстуры/хрупкий селектор).
- ❌ **Нет** — только вручную (внешняя авторизация, сеть, реальный git/AI, системные условия).

**Сводка:** всего кейсов — 38 · автоматизировано — 14 · доступно к автоматизации — 8 · только вручную — 16.

Как гонять автотесты — см. [maestro/README.md](maestro/README.md).
