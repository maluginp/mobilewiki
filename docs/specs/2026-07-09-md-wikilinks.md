# Спека: кликабельные [[wikilinks]] + эмбед картинок в просмотре md

## Решения
- **Резолвер (чистая логика, commonMain, тестируемо).** `VaultFile(relPath, absPath, name)`;
  `VaultRepository.allFiles()` — рекурсивный обход vault (dot-каталоги пропускаем).
  `resolveWikiLink(target, files)`:
  - последний сегмент имеет расширение → берём как есть, иначе добавляем `.md`;
  - есть `/` → матч по `relPath == нормализованный путь`;
  - иначе → матч по имени в любой папке, приоритет меньшей глубине (корень раньше),
    тай-брейк по relPath; не найдено → `null`.
- **Разбор контента (чистая логика).** `renderNote(content, files)` → блоки
  (`Text(md)` | `Image(absPath)`) + список `linkTargets`:
  - `[[t]]`/`[[path/t]]`/`[[t.md]]` найдено → в тексте заменяется на markdown-ссылку
    `[label](wikilink:<idx>)`, `label` = исходный target, `idx` → absPath в linkTargets;
  - `![[img.ext]]` c картинкой-расширением (png/jpg/jpeg/gif/webp/bmp/svg) и найденное,
    стоящее на отдельной строке → отдельный блок `Image(absPath)`;
  - не найдено (или эмбед не-картинка) → остаётся исходным текстом `[[...]]`/`![[...]]`.
- **Рендер.** MarkdownScreen рисует блоки: текст — `Markdown(...)` с переопределённым
  `LocalUriHandler` (схема `wikilink:` → `onOpenPath(target)`, прочее — дефолт);
  картинка — `Image` через `expect/actual decodeImage(bytes): ImageBitmap?`
  (androidMain: BitmapFactory), байты читает VaultRepository. Новых зависимостей нет.
- **Навигация.** `VaultViewModel.openPath(absPath)` — открыть файл по абсолютному пути
  (имя = из пути); back из заметки работает как раньше.

## Приёмочные тест-кейсы (ручной прогон)
- **[[file]] по имени.** Изн.: в vault есть `concepts/ai.md`, в открытой заметке текст
  `[[ai]]`. Шаги: открыть заметку → тап по `ai`. Ожидаемо: `ai` — ссылка, тап открывает
  `concepts/ai.md`.
- **[[path/file]] по пути.** Изн.: есть `docs/specs/x.md` и ещё `x.md` в другой папке,
  текст `[[docs/specs/x]]`. Шаги: тап. Ожидаемо: открывается именно `docs/specs/x.md`.
- **[[file.md]] с расширением.** Изн.: есть `README.md`, текст `[[README.md]]`. Шаги: тап.
  Ожидаемо: ссылка ведёт в `README.md`.
- **Ненайденная ссылка не кликабельна.** Изн.: текст `[[nope]]`, файла нет. Ожидаемо:
  отображается как обычный текст `[[nope]]`, не ссылка.
- **![[img.png]] картинка.** Изн.: есть `raw/pic.png`, текст `![[pic.png]]`. Ожидаемо:
  картинка отображается в просмотрщике.
