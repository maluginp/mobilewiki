# Описание шагов авторизации в GitHub

Jira: —  (tracker: none, free-text)
Branch: fix-github-auth-instructions

<intent>
На экране входа через GitHub (device flow) не хватает описания того, что именно
нужно сделать пользователю. Показать понятную инструкцию по шагам.
</intent>

<context>
Экран — `LoginScreen`, состояние `AuthState.AwaitingUser(userCode, verificationUri)`.
Сейчас в этом состоянии показаны: код, кнопка «Copy code», кнопка «Open GitHub»,
спиннер и «Waiting for confirmation…». Нет ни одного предложения, объясняющего
последовательность: скопировать код → открыть GitHub → вставить код → подтвердить.
Первый пользователь не понимает, что делать с кодом и кнопками.

Исходный текст задачи: «Не хватает описание что нужно сделать для авторизации в GitHub».
</context>

<requirements>
- В состоянии ожидания подтверждения показать пошаговую инструкцию, что сделать
  с кодом и кнопкой «Open GitHub», чтобы авторизоваться.
- Инструкция не должна появляться до старта авторизации (Idle).
</requirements>
